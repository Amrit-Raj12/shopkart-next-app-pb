# Shopkart

Welcome to Shopkart! This is a simple e-commerce platform built with Next.js.

## Installation

To install the necessary modules, run the following command:

```bash
npm install
```
## Start the project

To install the necessary modules, run the following command:

```bash
npm run dev
```
